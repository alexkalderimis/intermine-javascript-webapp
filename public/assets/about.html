Demonstration Web-App

This site is a test bed for the design process for the future of InterMine data-warehousing web-applications.

This site is entirely composed of a javascript application that communicates with an InterMine server using web-services. This allows us to prototype new interfaces quickly, and field-test the technologies that will be needed to support them.
